Biography

Zi Wang received his B.E. and M.E. degree at the School of Computer Science and Engineering in University of Electronic Science and Technology of China (UESTC). He is currently a PhD student in University of Exeter (UoE). His research interests focus on Mobile Edge Networks. He is also interested in high-performance computing, future Internet, cloud computing, Internet-of-Things and machine learning.

    • - Research Project: User Mobility Aware Task Assignment for Mobile Edge Computing
    Principal Researcher
    In this project, we jointly consider the task properties, the user mobility, and network constraints and aim at reducing the task execution delay by task scheduling in mobile edge computing. My contributions in this project are:
  • Literature review on task assignment methods in mobile edge computing.
  • Modelled the task assignment problem in mobile edge networks by jointly considering the user mobility, task properties and resource distributions.
  • Proposed a lightweight algorithm to the optimization problem.
  • Conducted simulation experiments for the proposed work.

    • - Research Project: Exploiting Mobile Sensor Data for Media Caching in Mobile Edge Networks
    Principal Researcher
    In this project, we propose a novel mobile sensor data based content caching framework, design a lightweight ambient data collection scheme and present an in-depth analysis on the sensor data for understanding and classifying user preferences. My contributions in this project are:
  • Literature review on sensors and caching schemes in mobile edge networks.
  • Designed and implemented a prototype to collect sensor data on Android-based smartphones.
  • Designed a sensor-data-aware framework for content caching in MEC.li>
  • Presented an in-depth analysis on the sensor data for understanding and classifying user preferences.
  • Designed and Implemented simulation experiments.

    • - Research Project: Highly-Efficient Bulk Data Transfer for Dissemination in Structured Wireless Embedded Network Systems
    Cooperative Researcher
    In this project, we propose reliable out-of-order transmission and bursty encoding mechanisms to transmit packets as many as possible in each transmission slot. The proposed transmission protocol can fully utilize each transmission slot and propagate data in the network as fast as possible. My contributions in this project are:
  • Literature review on dissemination protocols in wireless embedded networks.
  • Designed a novel full-slot reliable transmission mechanism (ULTRA).
  • Designed and Implemented the simulation experiment for ULTRA.

    • - Research Project: Position Free Keystroke Monitoring
    Cooperative Researcher
    In this project, we propose a geometrical approach to estimate the phone-keyboard relative position by combining the linguistic model and the Time Difference of Arrival, and we further propose a Position Free Keystroke Monitoring scheme based on the Mel-Frequency Cepstral Coefficient information and adaptive k-means clustering. My contributions in this project are:
  • Algorithm improvement for the relative position estimation.
  • Patent application.
